###概念
java虚拟机把.class文件加载到内存，并进行校验、解析、初始化，最终生成可被虚拟机识别的文件类型的过程
###生命周期
* 加载: 查找导入Class文件
* 链接
  * 验证: 检查文件正确性
  * 准备: 给类变量分配存储空间
  * 解析: 将符号引用转换成直接引用(偏移量等)
* 初始化: 初始化类变量
* 使用
* 卸载

###类装载
* 类加载器将.class文件的字节流读取到内存, 放在方法区, 并在堆中创建对应的java.lang.Class对象
* 类加载器有'启动类加载器' '扩展类加载器' '应用类加载器' '自定义类加载器', 采用双亲委托模式
